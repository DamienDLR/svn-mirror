l3obsolete
==========

This file lists expl3 commands that have been removed or 
replaced with respect to the version on the TeX Live 2009 disk.

==========

Removed				Replacement
-------				----------

The \cs_new* allocations have been renamed to \cs_new_local* and
what was \cs_gnew* is now \cs_new* as follows:

\cs_gnew_nopar:Npn                \cs_gnew_nopar:Npn
\cs_gnew:Npn                      \cs_gnew:Npn
\cs_gnew_protected_nopar:Npn      \cs_gnew_protected_nopar:Npn
\cs_gnew_protected:Npn            \cs_gnew_protected:Npn
\cs_gnew_nopar:Npx                \cs_gnew_nopar:Npx
\cs_gnew:Npx                      \cs_gnew:Npx
\cs_gnew_protected_nopar:Npx      \cs_gnew_protected_nopar:Npx
\cs_gnew_protected:Npx            \cs_gnew_protected:Npx
\cs_gnew_nopar:cpn                \cs_gnew_nopar:cpn
\cs_gnew:cpn                      \cs_gnew:cpn
\cs_gnew_protected_nopar:cpn      \cs_gnew_protected_nopar:cpn
\cs_gnew_protected:cpn            \cs_gnew_protected:cpn
\cs_gnew_nopar:cpx                \cs_gnew_nopar:cpx
\cs_gnew:cpx                      \cs_gnew:cpx
\cs_gnew_protected_nopar:cpx      \cs_gnew_protected_nopar:cpx
\cs_gnew_protected:cpx            \cs_gnew_protected:cpx

\cs_gnew_eq:NN  \cs_new_eq:NN
\cs_gnew_eq:Nc  \cs_new_eq:Nc
\cs_gnew_eq:cN  \cs_new_eq:cN
\cs_gnew_eq:cc  \cs_new_eq:cc


\num... functions		\int_... functions

\file_add_path:nN		None: l3file has been rewritten so that the
\file_input_no_record:n		      entire interface is at a higher level
\file_input_no_check:n
\file_input_no_check_no_record:n
\file_list_full:

\iow_space:			\c_space_tl as a generic explicit space
\msg_space:
\msg_two_spaces:
\msg_four_spaces:

\c_iow_comment_char	\iow_char:N \%
\c_iow_lbrace_char	\iow_char:N \{
\c_iow_rbrace_char	\iow_char:N \}

\msg_new:nnnnn			None: no longer include "recovery code" in message system.
\msg_set:nnnnn			Uses versions without "recovery code" argument, and code
\msg_generic_new:nnnn		recovery in place instead.
\msg_generic_set:nnnn
\msg_direct_interrupt:xxxxn
\c_msg_code_prefix_tl

\msg_kernel_new:nn		None: introduced new concept of "division" for kernel messages.
\msg_kernel_set:nn		(See also changes to :nnn functions.)

\exp_arg:x  -  This (expandable) function was only defined under LuaTeX.
               Use the unexpandable \use:x instead, which one day may become
               expandable after \exp_arg:x.
               
\num_elt_count:n			\tl_elt_count_aux:n (an internal function)
